Desde entonces he ido construyendo mi stack: HTML, CSS y JavaScript como base, Bootstrap y Tailwind para interfaces rápidas y bien diseñadas, y React junto con TypeScript para proyectos más escalables. Me importa entender el por qué detrás de cada herramienta que uso, no solo saber usarla
